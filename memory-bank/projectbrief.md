# K-Pop Accent Web App: Project Brief

## Project Overview

The K-Pop Accent Web App is a proof-of-concept interactive visualization of academic research on K-pop idols' English accents. The project transforms the findings from the paper "I Want to Be Born with That Pronunciation": Metalinguistic Comments About K-Pop Idols' Inner Circle Accents by Jihye Kim and Luoxiangyu Zhang into an accessible, engaging digital experience.

## Core Purpose

**Making Research Accessible**: This project aims to break down barriers to academic knowledge by presenting research findings in a visually appealing, interactive format that doesn't require specialized knowledge of research methods or academic conventions.

**Interactive Visualization**: Transform static research data into dynamic, interactive visualizations that allow users to explore the findings in ways that traditional reading cannot facilitate.

**Proof of Concept**: Demonstrate how academic research can be made more engaging and accessible through web technologies, serving as a template for future research visualization projects.

## Key Features (Simplified Scope)

1. **Video Analysis Hub**
   - Embed the 4 YouTube videos analyzed in the study
   - Display metadata from Table 1 for each video
   - Show comment distribution statistics

2. **Idol Profile Database**
   - Create profile pages for the 7 idols featured in the study
   - Present demographic background and accent information from Table 2
   - Include visual representations of comments about each idol

3. **Accent Sentiment Analyzer**
   - Interactive visualization of the 88.1% positive, 10.6% neutral, 1.3% negative distribution
   - Ability to filter comments by sentiment
   - Display examples of each sentiment type

4. **Comment Explorer**
   - Searchable interface for the 602 metalinguistic comments
   - Filter system based on categories from the research
   - Display of original language comments (Korean and English)

5. **Theoretical Framework Guide**
   - Educational content explaining key linguistic concepts
   - Interactive world map showing Kachru's three circles of English
   - Glossary of key terms used in the research

6. **Multi-Dimensional Analysis Tool**
   - Visualization showing relationships between sentiment and evaluation aspects
   - Exploration of correlations between different factors

## Success Metrics

1. **Accessibility**: Can users without academic backgrounds understand the key findings?
2. **Engagement**: Do users explore multiple features and spend time with the content?
3. **Visual Appeal**: Is the presentation visually attractive and intuitive to navigate?
4. **Academic Integrity**: Does the presentation maintain the accuracy of the research?
5. **Technical Performance**: Does the application function smoothly across devices?

## Development Approach

The project will be developed in two phases:

### Phase 1: Foundation (Independent of Comment Data)
- Project setup and design system implementation
- Video hub with metadata from the paper
- Idol profiles based on Table 2
- Educational content on linguistic concepts
- Research overview with placeholder visualizations

### Phase 2: Data Integration (When Comment Data Available)
- Data structure assessment and processing
- Comment explorer implementation
- Multi-dimensional analysis completion
- Feature integration and cross-navigation

## Project Constraints

- **Simplicity**: As a proof of concept, focus on core functionality over feature completeness
- **Academic Accuracy**: Maintain the integrity of the research findings
- **Accessibility**: Ensure the application is usable by a wide audience
- **Visual Design**: Implement the design system consistently across features
- **Performance**: Optimize for speed and responsiveness

## Target Audience

1. **Academic Researchers**: Looking for new ways to present their findings
2. **K-Pop Fans**: Interested in linguistic aspects of idol culture
3. **Language Enthusiasts**: Curious about accent perception and linguistic stereotyping
4. **Educational Users**: Students and educators in linguistics and cultural studies
5. **General Public**: Anyone interested in the intersection of language and pop culture

## Timeline

This is a proof-of-concept project with a flexible timeline, focusing on quality of implementation over speed of delivery.
